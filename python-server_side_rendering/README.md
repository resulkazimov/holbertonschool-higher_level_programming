# Python Server-Side Rendering

This project focuses on creating simple templating systems and exploring how data is dynamically injected into text and HTML files.

## Tasks

### 0. Creating a Simple Templating Program
A Python function `generate_invitations` that:
- Reads a template string.
- Processes a list of attendees.
- Handles missing data by replacing it with "N/A".
- Outputs personalized invitation files.
